# Grocery Management System

This project was developed as part of the **CSE370: Database Systems** course. It is a web-based grocery management system where an admin can view and manage data related to employees, customers, inventory, items, expiry tracking, and payment methods.

The system was built using **PHP**, **MySQL**, **HTML**, **CSS**, and some **JavaScript**, and is designed to run locally using **XAMPP**.
